# Apple M chip setup/deployment guide
1. install iTerm2
2. install brew
	a. install git `brew install git`
	b. install tmux `brew install tmux`
	c. install vim `brew install vim`
3. configure vim
	a.setup vundle
 
# Tools to Configure
- [Brew](brew.sh)
- [Tmux](https://github.com/tmux/tmux)
